# Selenium scraper
## Installation
 - make sure you have [Firefox](https://www.mozilla.org/en-US/firefox/new/) installed
 - install required Python packages:
 ```bash
 pip install -r requirements.txt
```
 - start the script:
 ```bash
 python scraper.py
```
